# Panduan Pengguna (User Guide)

Berikut adalah langkah-langkah untuk mengoperasikan sistem SIPS dengan benar.

## 1. Menyiapkan Bidang Baru
Jika ingin menambah bidang baru (misal: "Komite Audit"):
1. Buka menu **Master Bidang**.
2. Klik **Tambah Bidang**.
3. Masukkan nama dan pilih tema warna (misal: Emerald).
4. Simpan. Bidang baru akan langsung muncul di filter seluruh aplikasi.

## 2. Mengisi Data Master
Setelah membuat bidang, Anda wajib mengisi data pendukungnya agar bisa melakukan registrasi surat:
1. **Jenis Dokumen**: Isi kode seperti SRT (Surat) atau KEP (Keputusan) untuk bidang tersebut.
2. **Kualifikasi**: Isi kategori utama (misal: TAG untuk Tanggapan).
3. **Sub Kualifikasi**: Isi kategori spesifik (misal: KH untuk Hukum).
4. **Master Tujuan**: Tambahkan daftar penerima rutin bidang Anda.

## 3. Registrasi Surat
1. Buka menu **Registrasi Surat**.
2. Klik **Registrasi Surat Baru**.
3. **PILIH BIDANG TERLEBIH DAHULU**: Tunggu loading "Sinkronisasi" selesai.
4. Pilih Jenis Dokumen, Kualifikasi, dan Sub Kualifikasi.
5. Masukkan Tujuan (bisa pilih dari master atau ketik manual).
6. Masukkan Perihal.
7. Klik **Simpan Registrasi**. Nomor surat akan otomatis tercetak di tabel.

## 4. Tips Pencarian Data
Gunakan kotak pencarian di atas setiap tabel. Anda bisa mencari berdasarkan:
- Nomor Surat (sebagian atau lengkap)
- Perihal
- Nama Tujuan
- Kode Kualifikasi

Pencarian dilakukan secara instan seiring Anda mengetik.
